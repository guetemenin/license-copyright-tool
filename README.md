## ğŸ“œ Supported Licenses

This tool supports the following licenses:
- **Permissive Licenses**: MIT, Apache-2.0, BSD (2-Clause, 3-Clause), ISC, Zlib, etc.
- **Copyleft Licenses**: GPL-2.0, GPL-3.0, AGPL-3.0, LGPL-2.1, LGPL-3.0.
- **Creative Commons**: CC0-1.0, CC-BY-4.0, CC-BY-SA-4.0.
- **Specialized Licenses**: SIL-OFL, MPL-2.0, EUPL-1.1, NCSA.
